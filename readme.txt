== Description ==
Esay Full Responsive FAQ Accordion, just go to Accordion post, and add new Accordion. and use this shortcode [cb_faq] in any pages or posts. then enjoy it.

* Documentations: shortcode is [cb_faq].

* if you want to use in any template just use this code <?php echo do_shortcode('[cb_faq]'); ?>

* Default view latest 10 post.

* Show Manually Post Count of 5 post (just add count="5"). full code cb_faq [cb_faq count="5"] with third bracket.

* and support all post type, just use posttype="your post name"
